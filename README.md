# sealisp
`sealisp` is a small Lisp interpreter written in C.

## Features
- REPL with multi-line input
- `quote`, `if`, `define`, `lambda`, `begin`
- Arithmetic, comparisons, lists, `equal?`

## Building From Source
```bash
git clone https://github.com/Foox-dev/sealisp
cd sealisp
make
# Binary placed in build/
```
